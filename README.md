# life-clock-legal

Privacy policy, terms of use, and support pages for the Life Clock iOS
app, served via GitHub Pages.

Live URLs (after Pages is enabled):

- Privacy policy: https://kashane1.github.io/life-clock-legal/privacy-policy.html
- Terms of use: https://kashane1.github.io/life-clock-legal/terms-of-use.html
- Support: https://kashane1.github.io/life-clock-legal/support.html

The source markdown for these pages lives in the main repo at
`docs/products/life-clock/legal/` &mdash; this repo is just the published
HTML version that App Store Connect can link to.
